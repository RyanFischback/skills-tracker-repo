# Skills Tracker Repo

Public skills tracker for recurring weekly and monthly updates.

## Live Site

Once GitHub Pages is enabled for this repository, the site is available at:

- `https://ryanfischback.github.io/skills-tracker-repo/`

## Content Sources

- `skills/skills-directory.md`
- `skills/updates/*.md`

The web UI consists of:

- `index.html` (landing page)
- `viewer.html` (markdown viewer for skill docs and updates)
