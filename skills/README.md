# Skills Tracker Workspace

This directory is the GitHub source of truth for your evolving skills tracker.

## Files

- `skills-directory.md`: Core inventory + update cadence.
- `updates/weekly-template.md`: Copy for each week.

## Suggested Workflow

1. Update `skills-directory.md` when skill definitions or setup info changes.
2. Create weekly update files from template.
3. Commit updates in small PRs so progress is versioned and reviewable.
4. Optionally mirror this content into a shared Google Doc for non-technical viewers.
